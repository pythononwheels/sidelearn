Installieren

So holst du Learny auf den Homescreen

Learny ist eine installierbare Web-App (PWA) — kein App-Store nötig. Einfach eine Webseite, die direkt auf deinem Gerät läuft.

iPhone / iPad Dein Gerät
  1. Unten auf „Learny jetzt öffnen" tippen (in Safari).
  2. Unten auf Teilen tippen — das Quadrat mit dem Pfeil nach oben.
  3. „Zum Home-Bildschirm" wählen und Hinzufügen tippen. Fertig. 🥒
Android Dein Gerät
  1. Unten auf „Learny jetzt öffnen" tippen (in Chrome).
  2. Oben rechts auf das ⋮-Menü tippen.
  3. „App installieren" bzw. „Zum Startbildschirm hinzufügen". Fertig. 🥒

Tipp: Auf dem Desktop kannst du Learny direkt im Browser nutzen.
